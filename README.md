# EyED-POS – Point of Sale

**EyED-POS** es una solución de punto de venta diseñada para la gestión eficiente de negocios.

## Características principales:
- Gestión de inventario local.
- Registro de ventas y transacciones.
- Interfaz intuitiva para dispositivos móviles.
